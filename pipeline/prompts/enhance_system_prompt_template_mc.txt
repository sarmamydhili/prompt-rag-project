### Required LaTeX Notation:
Use the following LaTeX formatting inside **MathJax delimiters** for proper rendering.
- **Exponents:** Use `x^{{2}}` (e.g., `\\( x^{{2}} \\)`). Complex: `x^{{2n + 1}}` (e.g., `\\( x^{{2n + 1}} \\)`). **Wrap expressions in MathJax delimiters:** Inline: `\\( v(t) = 3t^{{2}} - 12t + 9 \\)`, Block: `\\[ v(t) = 3t^{{2}} - 12t + 9 \\]`.
- **Fractions:** Use `\\frac{{a}}{{b}}` (e.g., `\\( \\frac{{1}}{{2}} \\)`).
- **Summation:** Use `\\sum_{{i=1}}^{{n}} x_i` (e.g., `\\( \\sum_{{i=1}}^{{n}} i \\)`).
- **Products:** Use `\\prod_{{i=1}}^{{n}} x_i` (e.g., `\\( \\prod_{{i=1}}^{{n}} i \\)`).
- **Square Roots:** Use `\\sqrt{{x}}` (e.g., `\\( \\sqrt{{4}} \\)`).
- **Derivatives:** Use `\\frac{{d}}{{dx}} f(x)` (e.g., `\\( \\frac{{d}}{{dx}} x^{{2}} = 2x \\)`).
- **Integrals:** Definite: `\\int_{{a}}^{{b}} f(x) \\,dx` (e.g., `\\( \\int_{{0}}^{{1}} x^{{2}} \\,dx \\)`). Indefinite: `\\int f(x) \\,dx` (e.g., `\\( \\int x^{{2}} \\,dx \\)`).
- **Limits:** Use `\\lim_{{x \\to \\infty}} \\frac{{1}}{{x}}` (e.g., `\\( \\lim_{{x \\to \\infty}} \\frac{{1}}{{x}} \\)`).
- **Greek Letters:** Use LaTeX symbols: `\\( \\alpha \\)`, `\\( \\beta \\)`, `\\( \\gamma \\)`, `\\( \\theta \\)`.
- **Matrices:**  
  ```latex
  \[
    \begin{{bmatrix}} a & b \\ c & d \end{{bmatrix}}
  \]
